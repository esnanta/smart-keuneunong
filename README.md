# SMART KEUNEUNONG

Keuneunong adalah kalender tradisional Aceh sebagai panduan bertani dan melaut yang kini terancam punah akibat modernisasi dan perubahan iklim. Smart Keuneunong, aplikasi Android, mengintegrasikan kalender tradisional dengan data cuaca BMKG dan edukasi tanda alam. Digitalisasi ini menjadi strategi pelestarian, edukasi generasi muda, serta mitigasi bencana berbasis kearifan lokal.

## Minimum Viable Product (MVP)

Fitur inti yang akan tersedia pada versi pertama aplikasi (MVP) adalah:

1.  **Tampilan Kalender Keuneunong**: Menampilkan kalender tradisional dalam format digital yang mudah dibaca.
2.  **Integrasi Cuaca BMKG**: Menampilkan data cuaca saat ini dan prakiraan cuaca untuk lokasi pengguna.
3.  **Rekomendasi Pertanian & Perikanan**: Memberikan panduan dasar harian untuk kegiatan bertani dan melaut berdasarkan kalender Keuneunong.
4.  **Pusat Edukasi**: Menyajikan informasi singkat mengenai sejarah Keuneunong dan interpretasi tanda-tanda alam.
